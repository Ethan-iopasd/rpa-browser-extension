# RPA Flow v0.1.0-beta.1

## What it is

RPA Flow is a local browser automation workspace for Windows. This beta combines a React designer, Python agent, local FastAPI control plane, Tauri desktop shell, and optional recorder extension in one desktop-first package.

## Best for

- Internal tooling exploration
- Browser flow design and replay
- Desktop-local automation validation

## Quick start

Download and install `RPA Flow Desktop_0.1.0-beta.1_x64-setup.exe` from GitHub Releases, then launch the desktop app. If you want to record browser actions, also load `recorder-extension-<timestamp>.zip` in Chrome or Edge as an unpacked extension.

## Known limitations

- Windows-first beta release
- Experimental runtime and packaging
- Requires bundled local Chromium and Python sidecar components
- Compatibility is not yet guaranteed across future releases
