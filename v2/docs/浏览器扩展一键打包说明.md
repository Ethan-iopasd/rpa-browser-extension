# 浏览器扩展一键打包说明

## 一键命令

在 `v2` 目录执行：

```powershell
pnpm pack:recorder-extension
```

## 打包结果

- 输出目录：`v2/dist`
- 文件名格式：`recorder-extension-yyyyMMdd-HHmmss.zip`

## 说明

- 该命令会把 `v2/apps/recorder-extension` 目录内容直接打进 zip 根目录。
- zip 根目录下会直接包含 `manifest.json`，可直接用于扩展上传/分发。

## 可选参数（进阶）

如果你要自定义输出目录或文件名前缀，可直接调用脚本：

```powershell
powershell -ExecutionPolicy Bypass -File .\scripts\release\package_recorder_extension.ps1 -OutDir dist -NamePrefix recorder-extension
```

可选参数：
- `-SourceDir`：默认 `apps/recorder-extension`
- `-OutDir`：默认 `dist`
- `-NamePrefix`：默认 `recorder-extension`
